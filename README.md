## 🚀 Deployment (Encore Cloud)

Project Nexus has migrated to **Encore Cloud** for automated infrastructure and global scale.

1. **Install Encore CLI**: `curl -L https://encore.dev/install.sh | sh`
2. **Clone & Link**: `encore app link`
3. **Run Locally**: `encore run`
4. **Deploy**: `git push encore main`
